# voyager web framework

voyager is a web framework written in rust, inspired by Go `net/http`

## features

1. totally based on rust
2. clean api
3. minium dependency

## quick start

run the sample application:

```sh
cargo run --example hello
```

## contribute

before commit run:

```sh
cargo fmt
cargo clippy
```
